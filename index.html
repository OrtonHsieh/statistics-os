      紙 1 · 正面

      第 1 面｜選擇題 Router＋必錯陷阱

      目的：30 秒內辨識方法；承接 60% 選擇題。

      一定要抄

      ① 題型 Router

      先看 Y／問題類別 counts → χ²比較 3+ means → ANOVA數值 Y＋X → Regression依時間預測 → Forecastingrank／median／非常態 → Nonparametric

      ② 樣本結構決定工具

      1 類別變數 vs 指定比例χ² GOF
      2 類別變數是否有關χ² independence
      3+ 獨立平均數one-way ANOVA
      treatment＋blockrandomized block ANOVA
      1 X / 多 Xsimple / multiple regression
      paired、只看方向sign test
      paired、可排名差值Wilcoxon signed-rank
      2 independent rank groupsMann–Whitney
      3+ independent rank groupsKruskal–Wallis
      兩組排名一致性Spearman

      ③ 假設檢定固定骨架

      H₀ / H₁ → α → statistic＋df → p 或 critical → Reject / Do not reject H₀ → 回答商業問題

      用語：不拒絕 H₀ ≠ 接受 H₀；顯著關聯 ≠ 因果。

      ④ 高頻 MC 一行辨識

      • factor＝independent variable；treatment＝factor 的 level。
      • interaction：一個 factor 的效果隨另一 factor level 改變。
      • ANOVA 檢定 population means；F 顯著只代表至少一組不同。
      • SS 是平方和；MS=SS/df，不是一般 mean。
      • residual：e=y−ŷ；positive 表示實際值高於預測。
      • R²：Y 的 sample variation 被模型解釋比例，不是因果或準確率。
      • overall F：至少一個 βⱼ≠0；individual t：指定 βⱼ≠0。
      • 增加 X：R² 不降；adjusted R² 可能下降。
      • Forecast α 大／window 小 → 對新資料反應快、較不平滑。
      • χ²、ANOVA F 都是右尾；regression slope t 通常雙尾。
      • Nonparametric ≠ 無假設；paired 和 independent 不可混用。

      ⑤ df / Tail 快查

      χ² GOFk−1−m；右尾
      χ² independence(r−1)(c−1)；右尾
      ANOVAk−1, n−k；右尾
      Regression Fp, n−p−1；右尾
      Regression tn−p−1；通常雙尾
      Kruskal–Wallisk−1；右尾 χ²

      有空間再抄｜選項排除法

      先刪：單位錯、df 錯、把 sample 寫成 population、把「至少一組」寫成「全部」、把 association 寫成 causation 的選項。

      紙 1 · 背面

      第 2 面｜Regression 計算題 × 2

      Ch 14–16：斜率、R²、t/F、multiple regression、選模與診斷。

      最高優先

      ① Simple Regression 核心

      ŷ=b₀+b₁x
      b₁=Sxy/Sxx；b₀=ȳ−b₁x̄
      SST=Σ(y−ȳ)²
      SSR=Σ(ŷ−ȳ)²
      SSE=Σ(y−ŷ)²
      SST=SSR+SSE
      R²=SSR/SST=1−SSE/SST
      r=sign(b₁)√R²；e=y−ŷ

      b₁ 單位：Y 單位 / X 單位；X 增 1，預測 Y 平均改變 b₁。

      ② Regression ANOVA Table

      SourceSSdfMS / F
      RegressionSSRpMSR=SSR/p
      ErrorSSEn−p−1MSE=SSE/(n−p−1)
      TotalSSTn−1F=MSR/MSE

      Overall H₀: β₁=…=βₚ=0；Reject → 至少一個 slope 有效。

      ③ 係數推論

      s=√MSE；SE(b₁)=s/√Sxx（simple）
      tⱼ=bⱼ/SE(bⱼ), df=n−p−1
      CI: bⱼ ± tα/2,df·SE(bⱼ)

      pⱼ<α → 控制其他 X 後，Xⱼ 與 Y 有顯著線性關係。

      ④ 預測題工作流

      1. 寫完整 ŷ 式與單位。
      2. 代入每個 X；先乘後加。
      3. 報 point estimate。
      4. 若問 interval：mean response CI 比 individual prediction interval 窄。
      5. 檢查是否 extrapolation。

      ⑤ Multiple Regression

      ŷ=b₀+b₁x₁+…+bₚxₚ
      Adjusted R²=1−(1−R²)(n−1)/(n−p−1)

      bⱼ：其他 predictors 固定時，Xⱼ 增 1，ŷ 改變 bⱼ。

      Dummy：0 是 baseline；dummy coefficient＝相對 baseline 的 intercept 差。g 類需 g−1 個 dummy。

      ⑥ 新增變數 / Partial F

      F=[(SSEᵣ−SSE𝒻)/q] / [SSE𝒻/(n−p𝒻−1)]
      q=p𝒻−pᵣ

      Reduced SSE 一定 ≥ Full SSE。分母一定用 full model MSE。

      ⑦ Diagnostics

      Residual vs fitted 漏斗變異數不等
      U / 曲線漏 x² / 非線性
      時間殘差連續同號正自相關
      大 VIF / X 高相關multicollinearity
      離群且高 leverage可能 influential

      DW d=Σ(eₜ−eₜ₋₁)²/Σeₜ²；d≈2 無自相關，d≈0 正，d≈4 負。

      ⑧ 計算題最後 4 句

      Equation → test statistic / p → statistical conclusion → business interpretation（含單位、holding others constant）。

      有空間再抄｜陷阱

      高 R² 不保證 model valid；整體 F 顯著不代表每個 X 顯著；觀察資料不證因果；不要把 Y/X 單位顛倒。

      紙 2 · 正面

      第 3 面｜Forecasting × 2＋Chi-square × 1

      Forecasting 題庫最高頻：trend、seasonal、moving average、exponential smoothing。

      一定要抄

      ① Forecast 方法

      Naive: Fₜ₊₁=Yₜ
      k-MA: Fₜ₊₁=(Yₜ+…+Yₜ₋ₖ₊₁)/k
      WMA: Fₜ₊₁=ΣwᵢYᵢ/Σwᵢ
      ES: Fₜ₊₁=αYₜ+(1−α)Fₜ

      • WMA 最大權重給最新 actual。
      • ES 用本期 actual 更新「下一期」forecast。
      • α 大：反應快、較不平滑；α 小：反應慢、較平滑。
      • MA 有 trend 時會 lag。

      ② Trend / Seasonal

      Trend: Tₜ=b₀+b₁t
      b₁=[nΣtY−ΣtΣY]/[nΣt²−(Σt)²]
      b₀=Ȳ−b₁t̄
      Multiplicative: Forecast=Tₜ×Seasonal index

      Seasonal index >1：該季高於趨勢；四季 indices 平均約 1（總和約 4）。Deseasonalized Y=Y/index。

      ③ Error 與選模

      eₜ=Yₜ−Fₜ
      ME=Σe/n（bias）
      MAE=Σ|e|/n
      MSE=Σe²/n
      RMSE=√MSE
      MAPE=Σ|e/Y|×100%/n

      同資料、同 forecast periods 比較；MAE/MSE 越小越好。MAPE 遇 Y=0 不能用。

      ④ Forecast 計算題檢查

      時間排序 → 標示 actual/forecast → 第一個可算期 → 權重方向 → 不偷用未來資料 → rounding 留到最後。

      ⑤ Chi-square Router

      Goodness-of-fit1 個類別變數；是否符合指定比例
      Independence2 個類別變數；是否有關聯

      輸入是 counts。數字型原始值也可能先分成類別後使用 χ²。

      ⑥ Expected Counts

      GOF: Eᵢ=npᵢ；uniform E=n/k
      Independence: Eᵢⱼ=(row total)(column total)/n
      χ²=Σ(O−E)²/E

      每個 cell 都要算；差越大 χ² 越大。期望次數原則上至少 5。

      ⑦ Hypotheses / df / Decision

      GOF H₀：population proportions 符合指定分配。df=k−1−m（m＝由樣本估計的參數數）。

      Independence H₀：兩變數 independent。df=(r−1)(c−1)。

      右尾：χ²>critical 或 p<α → Reject H₀。

      ⑧ χ² 作答表格

      先畫 O / E / (O−E)²/E 三欄（列聯表則每格在旁寫 E）。最後報 χ²、df、p/critical、decision、情境結論。

      陷阱：拒絕 independence 只代表 association，不代表 causation；percent 不可直接當 O。

      有空間再抄｜題庫代表數字

      Uniform 3 類、n=150 → E=50；O=(40,60,50) → χ²=4。這組數字可當考場自我檢查模板。

      紙 2 · 背面

      第 4 面｜Nonparametric × 2＋ANOVA × 1

      先由 paired / independent / groups / rank correlation 選方法，再寫排名步驟。

      一定要抄

      ① Nonparametric Router

      1 sample / paired，只看 +/−Sign test
      paired differences，使用 rankWilcoxon signed-rank
      2 independent groupsMann–Whitney U
      3+ independent groupsKruskal–Wallis H
      兩套 ranks 的關聯Spearman rₛ

      ② Sign Test

      1. 差值＝After−Before；0 / tie 刪除。
      2. n＝非 0 的 + 與 − 總數。
      3. H₀:p=.5；X＝較少符號數（雙尾）。
      4. large n：μ=n/2，σ=√n/2，z=(X±.5−n/2)/(√n/2)。
      5. 雙尾 p=2×tail。

      ③ Wilcoxon Signed-rank

      1. d=After−Before；刪 d=0。
      2. 對 |d| 由小到大排名；ties 用平均 rank。
      3. 放回正負號，算 T⁺、T⁻。
      4. test statistic T=min(T⁺,T⁻)。
      5. 查 n 的 critical；小 T 反對 H₀。

      需要 paired，且差值分配大致對稱。

      ④ Spearman

      dᵢ=rank X−rank Y
      rₛ=1−6Σd²/[n(n²−1)]

      同方向排名 → +；反方向 → −。有 ties 時先用 average rank；可用 rank 後的 Pearson correlation。

      ⑤ Mann–Whitney / Kruskal–Wallis

      Mann–Whitney：合併兩組排名，算 R₁、R₂。

      U₁=n₁n₂+n₁(n₁+1)/2−R₁
      U₂=n₁n₂−U₁；U=min(U₁,U₂)
      μU=n₁n₂/2；σU=√[n₁n₂(N+1)/12]

      Kruskal–Wallis：合併 k 組排名。

      H=[12/(N(N+1))]Σ(Rⱼ²/nⱼ)−3(N+1)
      df=k−1
      ties: C=1−Σ(t³−t)/(N³−N)；Hc=H/C

      ⑥ One-way ANOVA

      H₀: μ₁=…=μₖ
      SST=SSTR+SSE
      SSTR=Σnⱼ(ȳⱼ−ȳ)²
      SSE=ΣΣ(yᵢⱼ−ȳⱼ)²
      MSTR=SSTR/(k−1)
      MSE=SSE/(n−k)
      F=MSTR/MSE

      Treatmentdf=k−1
      Errordf=n−k
      Totaldf=n−1

      ⑦ Block / Post hoc

      Randomized block：SST=SSTR+SSBlock+SSE；df error=(t−1)(b−1)。先控制 block variation，再檢定 treatment。

      Equal n Fisher LSD=tα/2,dfE√(2MSE/n)
      Unequal n: LSD=t√[MSE(1/nᵢ+1/nⱼ)]

      先有 significant overall F，才做 protected LSD。|ȳᵢ−ȳⱼ|>LSD → pair different。

      ⑧ ANOVA 陷阱

      F 大＝between signal / within noise 大；within estimate of σ² 不受 means 是否相等影響；between estimate 在 H₀ false 時會膨脹；結論只能先寫「not all means equal」。

      有空間再抄｜Factorial

      分別檢定 Factor A main effect、Factor B main effect、A×B interaction。interaction 顯著時，主效果需搭配各 level 解讀。
